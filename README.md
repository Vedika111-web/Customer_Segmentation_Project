# Customer Segmentation Project

## 📌 Project Overview
This project performs customer segmentation using machine learning (K-Means clustering).

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## 📊 Steps Performed
- Data Cleaning
- Feature Selection (Income, Recency)
- Applied K-Means Clustering
- Data Visualization

## 📈 Output
- Customers grouped into different clusters
- Scatter plot visualization generated

## 💡 Key Insights
- Customers with higher income form separate clusters
- Recency helps identify active vs inactive customers

## 🚀 Conclusion
Customer segmentation helps businesses target the right customers effectively.
